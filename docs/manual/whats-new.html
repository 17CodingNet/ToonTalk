Recent changes to ToonTalk

ToonTalk is changing all the time. For the full gory details visit the list of changes to the implementation.

Highlights of recent changes

  • Numeric functions.
    Added function birds which can be used for advanced numeric calculations. ToonTalk numbers can be combined by adding, subtracting, multiplying, and dividing. (A limited ability to compute a number raised to another number was removed and replaced by the power function bird.) Function birds can be used to obtain random numbers, compute trigonometric functions, logarithms, conversion to integer values, and modulo arithmetic. Released 23 February.
  • Robot descriptions.
    Trained robots now display a much improved description of what they do as an enhanced "tool tip". Also robots dragged to a text editor (not all accept drops) results in text that includes the description of what they do. Released 23 February.
  • Clear all.
    Dusty the Vacuum can now clear everything from the workspace. Everything is still in Dusty to "spit" out again if needed. Released 23 February.
  • Access to saved programs and pages.
    Listings of programs and saved pages (accessible via the settings button ()) now supports any number of files and search facilities. Released 5 February.

Earlier changes were not recorded here.
